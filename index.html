Pokemon Organizer

  • Pokemon List

  • This webapp was built to provide a single page that lets trainers check off caught pokemon and provide information on where they can find remaining pokemon.

    In my mind, Veekun, Serebii, and PokemonDB are the leaders in pokedex information. The goal of this app isn't to compete with any of the big guys, but instead to extend what they are already providing. The location information provided was captured from serebii, but does not provide the in-depth details that serebii or veekun is providing.

    This is a mashup made by Tyler Dillerstone. Checkout this project on github :)

    Sources

    • Serebii.net for location information
    • pokemondb.net for sprite images
    • Veekun.com for being a fantastic resource for all things pokemon
